# proyecto-ada-optimizacion-discreta

create virtual enviroment
```sh
python3 -m venv venv
```

install modules
```sh
pip install -r requirements.txt
```

To run the app move to the flask_app folder
```
cd flask_app
python3 app.py
```

you can acces the app in your localhost url

> http://127.0.0.1:5000
